1. Распакуй архив.
2. Создай НОВЫЙ пустой репозиторий на GitHub.
3. Загрузи в корень репозитория ВСЕ файлы и папки из этого архива.
4. Открой вкладку Actions.
5. Дождись завершения Build APK.
6. Скачай артефакт app-debug.

Важно:
- загружай содержимое архива в корень репозитория
- не загружай сам zip
- не загружай все файлы внутрь отдельной папки
